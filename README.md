# DatoCMS `<VideoPlayer>` and Custom Mux Data Environment

A simple demo showing how to use DatoCMS's [`<VideoPlayer/>`](https://github.com/datocms/react-datocms/blob/master/docs/video-player.md) component along with a custom `envKey` to send data to another [Mux Data](https://www.mux.com/data) account.
